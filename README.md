# Drop Menu CSS

Menu déroulant réalisé uniquement en HTML et CSS, sans JavaScript.

## Fonctionnalités

- Menu horizontal con sotto-menu a tendina
- Transizioni fluide al passaggio del mouse
- Accessibile da tastiera (`:focus-within`)
- Layout Flexbox

## Aperçu

Ouvrir `index.html` dans un navigateur pour voir le résultat.
