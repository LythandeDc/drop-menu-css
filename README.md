# Drop Menu CSS

## English

Dropdown menus built entirely with HTML and CSS, no JavaScript required. Four different examples to learn from.

### Examples

| File | Description |
|------|-------------|
| `index.html` | **Basic** — Simple red dropdown, minimal code |
| `dark.html` | **Dark** — Dark theme with accent colors |
| `animated.html` | **Animated** — Slide-down animation with gradient navbar |
| `mega-menu.html` | **Mega Menu** — Multi-column panel layout |

### Features

- Pure HTML & CSS, zero JavaScript
- Keyboard accessible (`:focus-within`)
- Smooth transitions
- Flexbox layout

### Usage

Open any `.html` file in a browser.
